# libft
 
Re-code a set of the libc functions, as defined in their man. Library use only next external functions: write(), malloc() and free().

Functions present the same prototype and behaviors as the originals:
• memset
• bzero
• memcpy
• memccpy
• memmove
• memchr
• memcmp
• strlen
• strlcpy
• strlcat
• strchr
• strrchr
• strnstr
